# PL/SQL Practice Notes

This is my personal PL/SQL practice repository from a 28-session learning series.

I used it to practice core concepts with code, slide images, and handwritten notes.

## What Is Included

- `PLSQL_Code/` for practice scripts
- `Slides/` for session slide images
- `.docx` files for my personal notes

## Topic Groups

1. Fundamentals and control flow
   Sessions: 1, 4, 5, 6, 7
2. Cursors and composite variables
   Sessions: 8, 9, 10, 11
3. Procedures and functions
   Sessions: 12, 13, 14, 15, 16, 26
4. Packages and transactions
   Sessions: 17, 18
5. Triggers and instead-of triggers
   Sessions: 19, 20
6. Records, collections, and bulk collect
   Sessions: 21, 22, 23
7. Dynamic SQL and temporary tables
   Sessions: 24, 27
8. Extra practice files
   JSON functions, datetime, OOP, test/debug scripts
